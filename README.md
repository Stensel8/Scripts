# Scripts Repository

Quick installer scripts for my most used tools and software. Run directly or download and modify as needed.

Most scripts are changed to fit my own needs, but can always be easily modified for your own use.

## How to use
git clone https://github.com/Stensel8/scripts.git

cd scripts

./<script_name>_installer.sh

pwsh ./<script_name>_installer.ps1

View this article for more info: [Installing PowerShell on Linux](https://learn.microsoft.com/en-us/powershell/scripting/install/installing-powershell-on-linux?view=powershell-7.5)
